## 주제

**프론트엔드 개발자 경력직 채용정보** — 서울 소재 (판교 제외), 2-5년 경력, 주요 타겟 3-4년차

## 수집 대상 플랫폼

원티드, 잡코리아, 사람인, 로켓펀치, 리멤버, 그룹바이, 점핏

## 작업 순서

1. 한국 시간 기준 오늘 날짜를 확인하고, **7일 전 날짜(YYYY-MM-DD)**를 계산하라.
2. WebSearch와 WebFetch로 각 플랫폼에서 채용공고를 수집하라.
   - **검색 쿼리 필수 규칙**:
     - 모든 WebSearch 쿼리에 `after:YYYY-MM-DD`(7일 전 날짜)를 반드시 포함하라. 예: `after:2026-05-30`
     - 모든 WebSearch 쿼리에 `-마감 -채용완료 -지원마감 -closed -expired`를 반드시 포함하라
     - 예시: `site:wanted.co.kr 프론트엔드 개발자 서울 경력 after:2026-05-30 -마감 -채용완료`
   - WebFetch로 개별 공고 페이지 접근이 가능한 경우, 페이지 본문에서 마감 여부를 확인하라
3. **검색 결과 snippet 마감 감지**: WebSearch 결과의 제목·설명(snippet)에 아래 키워드가 포함된 항목은 수집 목록에 추가하지 말고 즉시 제외하라:
   - `[마감]`, `마감`, `채용완료`, `지원마감`, `마감일 경과`, `closed`, `expired`, `채용 종료`
4. 플랫폼별 수집 결과를 `platform_collection_result`로 기록하라:
   - `success`: 정상 수집
   - `failed`: 접근 제한·차단·오류 등으로 수집 실패
   - `skipped`: 설정 또는 인증 정보 부재로 스킵
   - 일부 플랫폼 실패 시 나머지 플랫폼 결과만 계속 처리하라
5. 각 공고에 대해 아래 조건으로 필터링하라:
   - **지역**: 서울 소재만 포함, 판교 제외
   - **경력**: 2-5년 경력직만 포함 (신입·인턴 제외)
     - 허용 표현: "경력", "미들", "3년 이상", "5년 이하", "2-5년" 등
     - 제외 표현: "신입", "인턴", "주니어", "리드/시니어 (단, 2-5년 명시 없는 경우)"
   - **근무형태**: 재택·하이브리드·오피스 모두 포함
   - **등록일**: 오늘 기준 7일 이내 공고만 포함. 날짜를 확인할 수 없는 공고는 `등록일: 확인 불가`로 표시하되, 검색 결과에 마감 징후가 없는 경우에만 포함하라
   - **직무**: 프론트엔드 개발자 포지션만 포함
6. 중복 공고 제거: 동일 회사명 + 직무명이 겹치면 하나만 유지하라.
7. 각 공고마다 아래 정보를 수집하라:
   - 플랫폼명 (`platform`)
   - 회사명 (`company`)
   - 직무명 (`title`)
   - 공고 URL (`url`)
   - 지역 (`location`)
   - 근무형태 (`work_type`)
   - 경력요건 (`experience_level`)
   - 기술스택 (`tech_stack`)
   - 마감일 (`deadline`, 없으면 null)
   - 등록일 (`posted_date`, ISO 8601 또는 "확인 불가")
   - 중복 여부 (`is_duplicate`)
8. 결과를 `raw/YYYY-MM-DD.md` 경로로 새 파일을 생성하고 커밋·푸시하라.
   - YYYY-MM-DD는 오늘의 한국 날짜
   - 커밋 메시지: `scrap: YYYY-MM-DD`

## 출력 파일 형식

```
# Raw — YYYY-MM-DD

## 플랫폼별 수집 결과
| 플랫폼 | 상태 | 공고 수 | 비고 |
|--------|------|---------|------|
| 원티드 | success | 5 | |
| 잡코리아 | failed | 0 | 접근 차단 |
| 사람인 | success | 3 | |
...

## 수집 공고

### 1. (회사명) — (직무명)
- 플랫폼: (platform)
- URL: (url)
- 지역: (location)
- 근무형태: (work_type)
- 경력요건: (experience_level)
- 기술스택: (tech_stack, 쉼표 구분)
- 마감일: (deadline 또는 미정)
- 등록일: (posted_date)

### 2. ...
```

## 금지 사항

- `after:YYYY-MM-DD` 날짜 필터 없이 WebSearch 실행 금지
- `-마감 -채용완료 -지원마감` 제외 키워드 없이 WebSearch 실행 금지
- 7일 이전 공고 포함 금지
- 판교 소재 회사 포함 금지
- 신입·인턴 공고 포함 금지
- 서울 외 지역 공고 포함 금지
- 중복 공고 중복 수집 금지
- **모집 마감·채용 완료된 공고 수집 금지** — 검색 결과 제목·snippet 또는 공고 페이지에 "마감", "[마감]", "채용완료", "지원마감", "closed", "expired", "채용 종료" 등이 표시된 경우 즉시 제외하라
- 마감일이 오늘 이전인 공고 포함 금지
