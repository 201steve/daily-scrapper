## 1. Microsoft Build 2026: Windows Agent Framework·Project Polaris·Azure Agent Mesh 발표
- 출처: https://aitoolsrecap.com/Blog/microsoft-build-2026-windows-agent-framework-wsl3-azure-mesh
- 발행: 2026-06-02
- 본문:
  Microsoft unveiled the Windows Agent Framework (WAF), an MIT-licensed set of libraries and services that abstract agent lifecycle management, allowing developers to build AI agents that run across local Windows machines, Windows 365 Cloud PCs, and Azure Arc-enabled edge devices.
  Project Polaris, Microsoft's own in-house AI coding model, will replace GPT-4 Turbo as the default engine for all GitHub Copilot subscribers starting in August 2026, following automatic migration with an optional three-month fallback period.
  Polaris is a mixture-of-experts architecture with specialized sub-modules tuned for different programming languages and frameworks, outperforming GPT-4 Turbo on HumanEval and MBPP benchmarks with particular gains in low-resource languages like Rust and Haskell.
  Azure Agent Mesh was announced as a control plane that federates agent execution across on-premises Windows servers, Windows 365 Cloud PCs, and Azure Arc-enabled edge devices, with GA targeted for Q4 2026.
  Microsoft also announced Foundry Local reaching general availability for Windows, macOS (Apple Silicon), and Linux x64, enabling full AI inference and agent execution on-device without cloud dependency.

## 8. Sakana AI, 멀티에이전트 오케스트레이션 플랫폼 'Fugu Beta' 공개
- 출처: https://www.startuphub.ai/ai-news/technology/2026/sakana-fugu-beta-opens
- 발행: 2026년 5월 (최근)
- 본문:
  Sakana AI launched the beta for its new commercial AI product called Sakana Fugu, a multi-agent orchestration system that hits state-of-the-art on SWE-Pro, GPQA-D, and ALE-Bench.
  Fugu is built on the Conductor model, a 7-billion-parameter RL-trained model accepted at ICLR 2026 that orchestrates a pool of frontier worker LLMs (including GPT-5, Claude Sonnet 4, and Gemini 2.5 Pro) by dividing problems into subtasks and assigning each to the best-suited model.
  The Conductor never solves problems itself — it designs communication patterns between worker models and prompt-engineers focused instructions to each worker, achieving 87.5% on GPQA Diamond versus GPT-5's 82.3%.
  Sakana AI, founded by former Google researchers David Ha and Llion Jones, develops generative AI models using evolutionary "AI breeding" techniques and achieved unicorn status with approximately a $2 billion valuation.
  Fugu represents the company's first general commercial product after a history focused on enterprise AI research and foundational model development.
