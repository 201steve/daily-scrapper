## 4. GitHub Copilot, 사용량 기반 AI 크레딧 과금 체계 전환 시행
- 출처: https://github.blog/news-insights/company-news/github-copilot-is-moving-to-usage-based-billing/
- 발행: 2026-06-01
- 본문:
  GitHub Copilot plans are transitioning to usage-based billing on June 1, 2026; usage-based billing for GitHub Copilot is now live for all users.
  One AI credit equals $0.01; credits are consumed based on token usage, including input, output, and cached tokens, according to the published API rates for each model.
  Copilot Pro+ costs $39/month and includes $39 in monthly AI Credits; Copilot Business costs $19/user/month with $19 in monthly AI Credits; Copilot Enterprise costs $39/user/month with $39 in monthly AI Credits.
  All features aside from code completions and Next Edit Suggestions, which remain unlimited, will be measured and billed in GitHub AI Credits.
